HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is a child element of the head and it defines the title for a web page.
  • The style element defines the css styles asocciated with a website.
  • The base element defines the base URL for a web page.
  • The script element embeds client-side scripts in the HTML document.
  • The meta element defines charset / character set.
  • The link element creates a connection to an external file.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * selector is special wild card symbol, or * is used to declare ALL of the elements on the page
  • Using text-align is one way to center a title or heading on a webpage.
  • The img selector assigns property values to any image element in the markup.
  • The class selector will assign every element on HTML with this class to gain the property assignments from the CSS.
  • The box-shadow CSS property adds shadow effects around an element's frame.

Git

  • git status: checks what branch we are currently on.
  • git clone: repository clonning from GitHub to Git.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds modifications in the current branch to the staging area.
  • git commit -m "message": commit commant with a message that describes what the commit contains.
  • git diff --check: white space errors check.
  • git pull origin main: pull to recieve a branch's modifications into the local environment.
  • git push origin branch-name: push the changes made locally to the remote GitHub branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • console.log: method to output a message to the web console by adding an argument.
  • var keyword: to decalare a variable in JavaScript.
  • assignment operator =: to assign the specific data that we want stored in our variable.
  • When using the if statement: code will only execute if the statement meets a condition, or is true.
  • else if:To add more than one condition to a statement
  • The strict equality operator (===): checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
  • The function is set of instructions to tell the computer how to perform a certain task. To create a function, we use the function keyword.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
